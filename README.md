# Khadas Software Neural Network (Python API)

## Install

1. Install pip

```sh
$ sudo apt update
$ sudo apt install python3-pip
```

2. Install KSNN

```sh
$ pip3 install ksnn/ksnn_vim4-1.4-py3-none-any.whl
```
